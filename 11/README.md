# maman 11
In this maman we we wrote the command `ps` on xv6 system using a custom syscall.
Following this [guide](https://github.com/raj-maurya/xv6-public_modifiedOS) 

## How to run?
```bash
make clean
make qemu
```

## Expected Output:
```bash
name  PID    state 
  sh    2    Sleeping 
init    1    Sleeping 
  ps    3    Running  

```
